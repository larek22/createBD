# Legal RAG Studio (минимальная версия)

Простое настольное приложение для индексации текстовых правовых документов и полнотекстового поиска по ним. Проект состоит из двух частей:

* **FastAPI backend** — принимает путь к файлу, режет документ на статьи и сохраняет данные в локальный JSON.
* **PySide6 GUI** — в пару кликов отправляет файлы на индексацию, выполняет поиск и отображает отчёт проверки качества.

## Быстрый запуск на Windows

```bat
python -m venv .venv
.\.venv\Scripts\activate
pip install -r legal_rag_gui\requirements.txt
uvicorn legal_rag_gui.backend.server:app --host 127.0.0.1 --port 8765
```

В новом окне терминала:

```bat
.\.venv\Scripts\activate
python -m legal_rag_gui.main
```

## Батник (автозапуск)

Файл `run_legal_rag_studio.bat` создаёт виртуальное окружение, устанавливает зависимости, поднимает бэкенд и только после успешной проверки `/health` запускает GUI. Все сообщения сервера пишутся в `backend.log`.

## Возможности

* Индексация `.txt` и `.rtf` документов.
* Автоматическое разбиение по статьям и сохранение краткого описания.
* Поиск по ключевым словам и подсветка найденных статей.
* Простой отчёт качества: число статей, главы и части.
* Лог действий внутри приложения и файл `logs/backend.log` для диагностики.
