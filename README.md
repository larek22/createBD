# Legal RAG Studio

Эта репозитория содержит настольное приложение **Legal RAG Studio** для создания и тестирования юридических векторных баз знаний. Проект состоит из графического клиента на PySide6 и backend-сервера на FastAPI, которые работают совместно с Qdrant и моделями OpenAI.

## Запуск на Windows

Для простого старта добавлен скрипт [`run_legal_rag_studio.bat`](./run_legal_rag_studio.bat). Он автоматически:

1. Проверяет наличие Python 3.x.
2. Создаёт (или переиспользует) виртуальное окружение `.venv` в корне репозитория.
3. Устанавливает зависимости из `legal_rag_gui/requirements.txt`.
4. Запускает backend на Uvicorn.
5. Открывает графический интерфейс Legal RAG Studio.
6. По закрытии окна останавливает backend.

### Как использовать

1. Скачайте или клонируйте репозиторий в удобную папку.
2. Дважды щёлкните по файлу `run_legal_rag_studio.bat` (или запустите его из командной строки `cmd`).
3. Дождитесь установки зависимостей при первом запуске.
4. После появления окна приложения можно загружать документы, строить базу и выполнять поиск.

> ⚠️ Скрипт читает порт backend из пользовательского конфига (`~/.legal_rag_studio/config.yaml`). Если конфиг отсутствует, используется порт `8765`.

> ℹ️ Начиная с этой версии, сам GUI тоже умеет запускать bundled backend. Поэтому если вы стартуете `python -m legal_rag_gui.main` вручную, сервер FastAPI появится автоматически — достаточно подождать сообщение «✅ Backend запущен…» в нижней панели логов.

## Ручной запуск (альтернатива)

Если требуется запуск вручную:

```bash
python -m venv .venv
.\.venv\Scripts\activate
pip install -r legal_rag_gui\requirements.txt
uvicorn legal_rag_gui.backend.server:app --host 127.0.0.1 --port 8765
```

Затем в новом окне терминала (с активированным `.venv`):

```bash
python -m legal_rag_gui.main
```

## Структура проекта

- `legal_rag_gui/` — исходный код приложения.
